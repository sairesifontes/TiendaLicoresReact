# Licores Altagracia
    Project for CoderHouse course, using React js, Tailwind, Firebase.

## Versiones
- v18.17.1 node
- v10.2.5 npm

## Start Project
- npm run start

## Deploy
- npm run build
- npm install -g serve
- serve -s build
